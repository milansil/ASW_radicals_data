Title: Assessing realistic binding energies of some essential interstellar radicals with amorphous solid water: A fully quantum chemical approach
Author(s): M. Sil, A. Roy, P. Gorai, N. Nakatani, T. Shimonishi, K. Furuya, N. Inostroza-Pino, P. Caselli, A. Das
===========================================================================================================================================================

Description of contents: 
This tar.gz archive file contains 9 subdirectories:

  ASW_clusters: Contains 9 amorphous solid water (ASW) ice structures, each containing 20 H2O molecules.
  Radicals: Contains the 7 diatomic radical structures.
  CH+ASW_1: Contains the CH radical + ASW_1 complex structures in 7 different dangling-H sites of ASW_1.
  HS+ASW_1, NH+ASW_1, NO+ASW_1, NS+ASW_1, and CN+ASW_1: Similar to CH+ASW_1 as described before, but for the other radicals.
  OH+ASW_clusters: Contains 9 more subdirectories inside for the OH+ASW complex structure for all the 9 different ASW clusters, each containing 20 H2O molecules.

Each subdirectory contains '.xyz' formatted files (the unit used for the coordinates is Angstroms) for the species studied in the accepted version of the manuscript. The charge and multiplicity of different species are also included.

**File Format:** '.xyz'
**Coordinate System:** Right-handed Cartesian coordinates (Angstroms)
**Software Compatibility:** The files can be read using a chemical visualization tool such as Chemcraft or any other. More info about the .XYZ format can be found here: https://en.wikipedia.org/wiki/XYZ_file_format
===========================================================================================================================================================

Example: For NO Radical


 0 2
  N                  0.00000000    0.00000000   -0.60941000
  O                  0.00000000    0.00000000    0.53323400


Here 0 is the charge, 2 is the multiplicity of the NO radical.

The column values are the cartesian coordinates for the atoms in the radical.
